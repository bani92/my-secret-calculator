대시보드 월 선택 UX

등록된 월을 선택해 상세 대시보드 보기

세 시안 모두 등록된 월을 먼저 보여주고, 월을 클릭하면 현재 기획된 월별 요약, 카테고리 지출, 지출 목록을 같은 화면에서 갱신하는 흐름입니다.

시안 A. 상단 월 칩 + 하단 상세

현재 대시보드 구조를 가장 적게 흔드는 방식입니다. 월이 많지 않을 때 빠르게 스캔하기 좋습니다.

추천: 단순/빠른 적용

시안 B. 좌측 월 목록 + 우측 상세

등록 월이 늘어나도 안정적입니다. 데스크톱에서 월 선택과 상세 비교가 가장 명확합니다.

추천: 장기 사용

시안 C. 연간 월 보드 + 선택 상세

한 해의 흐름을 먼저 보고 특정 월로 들어가는 방식입니다. 월별 패턴을 보는 재미가 있습니다.

추천: 분석 중심
