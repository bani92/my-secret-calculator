목적
----
이 프로젝트는 개인용 로컬 가계부 웹 앱이다.

현재 1차 버전은 Vue 로컬 앱이며, 브라우저 localStorage에 데이터를 저장한다.
로그인, 클라우드 동기화, 은행/카드 자동 연동, 백엔드는 아직 범위 밖이다.

장기 방향:
1. 로컬 Vue 앱을 먼저 완성한다.
2. JSON 내보내기/가져오기로 백업과 수동 기기 이동을 지원한다.
3. 나중에 노트북/휴대폰 자동 동기화가 필요하면 Spring Boot + DB를 추가한다.
4. 진짜 자동 동기화는 휴대폰과 노트북이 모두 접근 가능한 서버/클라우드 저장소가 필요하다.

현재 앱 구조
------------
상단 탭은 현재 4개다.

1. 입력
   - 기본 시작 화면.
   - 월 수입 저장.
   - 일일 지출 입력.
   - 금액 입력은 콤마 표시.
   - 월 수입 저장 후 input 비움.

2. 대시보드
   - 시안 A 방향으로 반영됨.
   - 등록 년도 선택.
   - 선택 년도에 등록된 월 칩 표시.
   - 월 칩 클릭 시 기존 월 상세 내용 표시.
   - 월 수입, 지출, 남은 금액, 지출률 표시.
   - 카테고리별 지출과 월별 지출 목록 표시.

3. 통계
   - 안 A: 지출 흐름 중심으로 구현됨.
   - 년도별 총 지출 CSS 막대 그래프.
   - 선택 년도의 월별 지출 CSS 막대 그래프.
   - 선택 년도의 총 지출, 지출월 평균, 최고 지출 월 요약 카드 표시.
   - 외부 차트 라이브러리는 사용하지 않음.

4. 사람
   - 사람별 받을 돈/줄 돈 기록.
   - 정산 완료 기록은 히스토리에 남김.
   - 정산 완료 기록은 현재 미정산 잔액에서만 제외.
