현재 앱 구조
------------
상단 탭은 현재 4개다.

1. 입력
   - 월 수입 저장.
   - 일일 지출 입력.
   - 금액 입력은 콤마 표시.
   - 저장 후 input 정리.

2. 대시보드
   - 등록 년도 선택.
   - 선택 년도에 등록된 월 칩 표시.
   - 월 칩 클릭 시 기존 월 상세 내용 표시.
   - 월 수입, 지출, 남은 금액, 지출률 표시.
   - 카테고리별 지출과 월별 지출 목록 표시.

3. 통계
   - 년도별 총 지출 CSS 막대 그래프.
   - 선택 년도의 월별 지출 CSS 막대 그래프.
   - 총 지출, 지출월 평균, 최고 지출 월 요약 카드.
   - 최고 지출 월 동률이면 `4월 외 1개월 · 20,000원` 형태로 표시.
   - 외부 차트 라이브러리는 사용하지 않음.

4. 사람
   - 사람별 받을 돈/줄 돈 기록.
   - 정산 완료 기록은 히스토리에 남김.
   - 정산 완료 기록은 현재 미정산 잔액에서만 제외.
